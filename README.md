# 小剧场提示词库

SillyTavern 扩展插件，连接小剧场管理器后端，提供提示词浏览、搜索、随机抽取、关联保存等功能。

## 功能

- **提示词浏览** — 从小剧场管理器 API 获取提示词，支持列表查看和详情展示
- **搜索与标签筛选** — 按关键字搜索提示词，按标签精确筛选
- **随机抽取** — 在当前筛选范围内随机抽取指定数量的提示词
- **复制与插入** — 一键复制提示词全文，或直接插入/追加到聊天输入框
- **关联保存** — 在 AI 回复的消息上显示保存按钮，将角色回复保存到小剧场管理器，并可关联到已有提示词
- **智能匹配** — 保存角色回复时，自动分析输入框上下文，推荐最相关的提示词供关联

## 安装

1. 在 SillyTavern 的扩展面板中点击"安装扩展"
2. 输入本仓库 URL
3. 或在 `plugins` 目录下手动放置

## 配置

点击扩展按钮打开面板后，进入设置页：

| 配置项 | 说明 |
|--------|------|
| 管理器地址 | 小剧场管理器后端地址，默认 `http://localhost:7788` |
| 管理器密码 | 可选，如果后端开启了密码保护 |
| 随机数量 | 随机抽取时每次抽取的条数（最大 50） |

## 使用

1. 点击侧栏"小剧场提示词库"按钮打开面板
2. 浏览或搜索提示词
3. 点击"查看"查看完整内容并复制/插入
4. 使用标签筛选后点击"随机抽取"获取随机提示词
5. 鼠标悬停 AI 回复消息，点击出现的纸飞机图标即可保存该回复

## 依赖

- SillyTavern
- 小剧场管理器后端（提供 API）

## API

需要后端提供以下接口：

- `GET /api/data` — 获取提示词列表
- `POST /api/theaters` — 保存角色回复
- `POST /api/auth/login` — 密码认证获取 token
